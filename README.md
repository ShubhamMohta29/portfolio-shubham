# Portfolio — Shubham Mohta

My personal portfolio website built with Three.js, plain HTML, and CSS — deployed on Vercel.

## About

An interactive 3D galaxy portfolio where each section of the site is a stellar system you navigate to by clicking on nav stars in a spiral galaxy. Built from scratch with no frameworks.

## Built With

- Three.js (3D rendering)
- HTML5 / CSS3 / Vanilla JavaScript
- Vercel (hosting)

## Features

- 3D spiral galaxy with 3 arms, dense star field, galactic core glow, and halo
- Interactive nav stars — click to fly into a stellar system for each section
- Each section is a planetary system with orbiting planets you click to explore
- Background star field (12,000 stars) that persists across scene transitions
- Smooth camera transitions between galaxy view and stellar views
- Planet detail bottom sheet with project info, skills, links, and PDF résumé viewer
- Name and majors displayed at the top of the page in galaxy view

## Sections

- **About** — bio, background, skills, interests (planetary system)
- **Projects** — Sudoku Solver, Portfolio, H.A.D.E.S., Argus, Animal Encyclopedia
- **Contact** — email, GitHub, LinkedIn
- **Résumé** — education, skills, inline PDF viewer

## Running Locally

Requires a local server (ES modules don't work over `file://`):

```bash
git clone https://github.com/ShubhamMohta29/portfolio-shubham.git
cd portfolio-shubham
npx serve .
```

Then open `http://localhost:3000` in your browser.

# Portfolio- Shubham Mohta

My personal portfolio website built with plain HTML, CSS, and JavaScript — deployed on Vercel.

## About

A clean, minimal developer portfolio showcasing my projects, skills, and contact info. Features an animated particle network background built with the HTML5 Canvas API.

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Vercel (hosting)

## Features

- Animated particle network background (Canvas API)
- Responsive layout
- Smooth scroll navigation
- Skills section
- Project cards with GitHub and live demo links
- Contact section with email and social links

## Sections

- **Hero** — intro and call to action
- **About** — short bio and skills (AI Systems, Python, Java, Git)
- **Projects** — cards for each project with links
- **Contact** — email, GitHub, LinkedIn

## Live Site

[portfolio-shubham-gilt.vercel.app](https://portfolio-shubham-gilt.vercel.app)

## Running Locally

No build tools needed. Just clone and open:

```bash
git clone https://github.com/shubhammohta29/portfolio.git
cd portfolio
open index.html
```

Or drag `index.html` into your browser.

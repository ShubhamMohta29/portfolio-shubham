# Portfolio — Shubham Mohta

My personal portfolio website built with plain HTML, CSS, and JavaScript — deployed on Vercel.

## About

A clean, minimal developer portfolio showcasing my projects, skills, and contact info. Built with no frameworks or libraries.

## Built With

- HTML5
- CSS3
- Vercel (hosting)

## Features

- Flat, editorial design with a single accent color
- Responsive layout
- Smooth scroll navigation
- Skills section
- Project list with GitHub links
- Contact section with email and social links

## Sections

- **Hero** — intro and call to action
- **About** — short bio, photo, and skills (Python, Java, JavaScript, React, AI Systems, Git)
- **Projects** — linked list for each project (Sudoku Solver, Portfolio, H.A.D.E.S., Argus, Animal Encyclopedia)
- **Contact** — email, GitHub, LinkedIn

## Live Site

[portfolio-shubham-gilt.vercel.app](https://portfolio-shubham-gilt.vercel.app)

## Running Locally

No build tools needed. Just clone and open:

```bash
git clone https://github.com/shubhammohta29/portfolio.git
cd portfolio
open index.html
```

Or drag `index.html` into your browser.

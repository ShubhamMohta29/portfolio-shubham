Animal Enyclopedia

Java · Swing · REST APIs

A desktop animal encyclopedia app built collaboratively with a team of 6 at the University of Toronto.

My contributions focused on two areas:

  • Retrieve Info use case — I designed and implemented the full backend pipeline for animal search: the animal entity and animalFactory (JSON → object parsing with unit conversion for lifespan, weight, and height), the APIClass client that hits the API Ninjas Animals API, and the interactor/presenter/view model chain following Clean Architecture. The search flow handles single results, multiple matches (shown as a selectable list), and graceful error states.
  • GUI layer — I built several of the app's Swing screens including SuccesfulSearch (animal info display with favorite and trading card buttons), MultiSuccesfulSearch (disambiguation list when multiple animals match), and MainMenu (search bar, fuzzy suggestion links, and navigation to all features)
